# 股票多因子分析系统 — 技术文档

## 1. 项目概述

一个基于 Python 的 A 股多因子分析与历史形态匹配系统。通过免费数据源拉取历史日线和财报数据，计算 60+ 个量化因子，提供交互式 Dashboard 进行可视化分析，并支持基于 Pearson 相关系数的历史形态匹配策略。

**技术栈**: Python 3.9 + Streamlit + Plotly + AkShare + Pandas + NumPy + SciPy

## 3. 数据架构

### 3.1 数据源

| 数据层 | 来源 | API | 频率 | 备注 |
|--------|------|-----|------|------|
| 日线行情 | 新浪财经 | `stock_zh_a_daily` | 日频 | OHLCV + 换手率 + 流通股本 |
| 估值数据 | 百度股市通 | `stock_zh_valuation_baidu` | 双周频 → 日频 | PE(TTM)/PE(静)/PB/市现率/总市值 |
| 财务报表 | 同花顺 | `stock_financial_abstract_ths` | 季度 → 日频 | ROE/毛利率/增长率/资产负债率等 |
| 指数行情 | 新浪财经 | `stock_zh_index_daily` | 日频 | 用于计算 Beta |

### 3.2 价格修正机制

新浪前复权价格可能因近期除权除息滞后导致偏差。系统使用独立数据源进行交叉修正：

```
真实股价 = PB(百度) × BPS(同花顺财报)
```

- PB 来自百度双周频数据，线性插值为日频
- BPS 来自同花顺季报，前向填充到日频
- 计算日频真实价格后等比修正新浪 OHLC，保留成交量/换手率不变

### 3.3 数据缓存

- 所有 API 返回数据缓存到本地 `data/` 目录 (CSV 格式)
- Streamlit `@st.cache_data` 缓存因子计算结果 (TTL 1 小时)
- 形态匹配相关系数矩阵独立缓存，权重调整不触发重算

| 类别 | 因子 | 计算方式 |

## 6. 历史形态匹配 (核心策略)

### 6.1 策略原理

**历史类比法 (Historical Analogy / Pattern Matching)**

当前一段时间的因子曲线作为"模板"，在历史数据中滑动搜索形态最相似的窗口，然后观察那些相似窗口**之后**发生了什么，以此作为预测参考。

### 6.2 算法实现

**相似度度量**: Pearson 相关系数 (scipy.stats.pearsonr)
- 衡量形态方向一致性，不受绝对值差异影响
- 对每个选中因子独立计算滑动相关系数

**多因子综合评分**:
```
综合得分 = Σ(因子_i 的 Pearson r × 权重_i)
```
- 权重可配置，总和自动归一化为 1
- 按综合得分降序排列取 Top N

**搜索范围**: 排除模板自身所在位置，扫描全部历史有效数据

### 6.3 交互配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 匹配窗口 | 模板长度 (交易日) | 10 天 |
| 显示前 N 名 | 展示的匹配数量 | 5 |
| 预测展望 | 后续走势显示天数 | 与窗口同 |
| 模板截止日 | 模板窗口的结束日期 | 最新日期 |
| 匹配因子 | 用于匹配的因子 (多选) | close, PE, PB, RSI |
| 因子权重 | 可调滑块, 自动归一化 | 均分 |

### 6.4 可视化设计

- **当前模板图**: 所有选中因子叠加，X 轴为相对天数 (0=截止日)
- **排名分组**: 每个排名一行，每个因子一张小图，并排展示
- **每张小图**:
  - 蓝色实线 = 历史匹配段
  - 蓝色虚线 = 历史预测段
  - 红色实线 = 当前模板
  - 灰色点线 = 匹配/预测分界 (x=0)
- **共识预测图**: 所有历史预测叠加，均值线 + ±1σ 置信带
- **悬停提示**: 鼠标悬停显示实际日期和数值

## 7. 数据验证

运行 `python3 validate.py <股票代码>` 进行自动化数据校验：

- **时效性**: 检查日线/估值/财报最新日期
- **日线数据**: OHLC 大小关系、价格非负、涨跌幅计算
- **估值交叉**: PE/PB 范围、PB×BPS ≈ 价格
- **财务逻辑**: 毛利率 ≥ 净利率、流动比率 ≥ 速动比率等
- **技术指标**: RSI 范围、布林带顺序、MACD 合理性、Beta/波动率非负

## 9. 已知限制

1. **数据源依赖**: 新浪/百度/同花顺免费 API，稳定性受网络环境影响，部分接口可能偶发连接失败
2. **PE/PB 频率**: 百度估值数据为双周频，插值为日频后精度有限
3. **财报滞后**: 季报发布有约 1-2 个月延迟，最新财报可能不是最新季度
4. **Beta 数据**: 新浪指数数据可能滞后 1 年，Beta 对近期风险判断有偏差
5. **形态匹配局限**: Pearson 相关系数只衡量线性相关，无法捕捉复杂非线性形态；历史匹配不代表未来必然重复
